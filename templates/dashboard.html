API Endpoints

SMS & Communication

  • POST /api/sms/send
    Send SMS messages
  • GET /api/mock/sms/history
    Get SMS history
  • POST /api/mock/sms/simulate-incoming
    Simulate incoming SMS

Verification Services

  • POST /api/verification/create
    Create service verification
  • GET /api/verification/{id}/number
    Get verification number
  • GET /api/verification/{id}/messages
    Get received SMS codes
